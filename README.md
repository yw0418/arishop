## Description
Spring MVC Framework 기반의 개인 프로젝트입니다.
<br>회원가입, 로그인, 회원 탈퇴, 지도 검색, 상품 등록, 장바구니, 회원정보 수정, 상품 수정, 관리자 페이지 구현, 비밀글 작성 기능을 사용할 수 있습니다.
